Voor het vak Artifical Intelligence project heb ik met twee klasgenoten een zelfrijdende auto gemaakt. Omdat we door covid niet konden samenkomen hebben we dit in het spel Forza Horizon 4 gedaan ipv op een modelauto.

Zelfrijdende auto

Het doel van dit project was om de aangeleerde technieken toe te passen. We hebben dan voor FastAI en Python gekozen. Eerst zijn we begonnen met het analyseren hoe we dit gingen doen. We zijn gaan kijken hoe we de output van de game als input voor ons model gingen gebruiken. Hier hebben we dan een voor image regression gekozen. Onze data bestond uit een game-frame met de corresponderende controller input. Hierbij leerde we ons model aan om voor game-frames de controller input te predicten. Deze controller input stuurde we dan via een virtuele controller terug door naar de game.

Uit dit project heb ik vooral geleerd hoe belangrijk het is om op voorhand je opdracht goed te analyseren om zo direct in de juiste richting te beginnen. Het was ook belangrijk om goed te communiceren zodat iedereen wist wat hij moest doen.

Applicatie om spelfouten te vinden

Github

Voor een online cursus van Harvard heb ik een applicatie geschreven om spelfouten in een tekst te vinden. Ik heb ook getimed hoelang het duurde om al deze woorden te controleren.

Applicatie om spelfouten te vinden

In dit project was het de bedoeling om grote bestanden met tekst te controlleren op spelfouten. Zoals je hieronder kan zien duurde het 2,31s om de 17.756 woorden van de film La La Land te controlleren met de 143.091 woorden uit het gebruikte woordenboek. De moeilijkheid zit hier in de grote teksten. Als het programma niet efficient is dan kan het tot minuten duren tegen alle woorden zijn vergeleken. Hierdoor heb ik gekozen om te werken met een hash table waarvan elke waarde gelinkt was aan een linked list. Spijtig genoeg komt dit allemaal niet aan bod in onze opleiding. Deze applicatie is volledig in de programmeertaal C geschreven.

Ik heb geleerd hoe belangrijk het is om efficiente programma's te schrijven. Ook heb ik geleerd hoe een linked list werkt en hoe ik deze moet implementeren.

#4 HTF 2020 - Deep Learning & NLP

Ik heb met een vriend meegedaan aan de Hack The Future hackathon van de Cronos groep. We deden mee aan de Deep Learning & NLP challange. Deze challange werd gecoached door personen van Brainjar. Bij deze hackathon was de opdracht om in een groep van 2 trolls te vinden in een bepaald forum. De data bestond uit forum-posts en comments. Met FastAI hebben we 2 textclassifiers getrained, één voor de categorieën en één voor sentiment. We hebben de forum-posts gefiltered op categorie en hebben sentiment geclassified op de comments. Als een post veel negatieve comments had, was het een troll post.
