Mouth animation

What was the goal of this internship? To make a model that generates mouth movements when given a image or video and a audio sample. The model learns the mouth positions for every sound wave.

Because there is this fantastic work of Wav2Lip I decided to build on top of this library instead of starting from scratch. The extensions that I made are: the original model is trained on English data, so I'll be fine-tuning on Dutch data. This data needed to be scraped and pre-processed because there were no existing datasets available. The other extension is to make it possible when you use a group picture that you can choose for which person you want to generate the mouth animation. In the original model it generated them for the first person it detected.

Nim AI

Github

A NIM AI that uses learns how to play trough reinforcement learning.

Nim AI

Het doel van deze AI is het implementeren van Q-learning. Dit algoritme speelt veel spellen tegen zichzelf en update elke keer de waarden van de Q-table. Dit zorgt ervoor dat de AI over tijd beter en beter wordt. Dit heeft als effect dat na een bepaalde tijd het onmogelijk is om nog te winnen van de AI omdat hij zo goed is geworden. The goal is to implement Q-learning. The algorithm plays a lot of games against itself and updates everytime the values of a Q-table. Because of this the AI gets better over time. After some time it's impossible to win against the AI as a human because it got so good.
