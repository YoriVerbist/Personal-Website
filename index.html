Applicatie om spelfouten te vinden

Github

Voor een online cursus van Harvard heb ik een applicatie geschreven om spelfouten in een tekst te vinden. Ik heb ook getimed hoelang het duurde om al deze woorden te controleren.

Applicatie om spelfouten te vinden

In dit project was het de bedoeling om grote bestanden met tekst te controlleren op spelfouten. Zoals je hieronder kan zien duurde het 2,31s om de 17.756 woorden van de film La La Land te controlleren met de 143.091 woorden uit het gebruikte woordenboek. De moeilijkheid zit hier in de grote teksten. Als het programma niet efficient is dan kan het tot minuten duren tegen alle woorden zijn vergeleken. Hierdoor heb ik gekozen om te werken met een hash table waarvan elke waarde gelinkt was aan een linked list. Spijtig genoeg komt dit allemaal niet aan bod in onze opleiding. Deze applicatie is volledig in de programmeertaal C geschreven.

Ik heb geleerd hoe belangrijk het is om efficiente programma's te schrijven. Ook heb ik geleerd hoe een linked list werkt en hoe ik deze moet implementeren.
